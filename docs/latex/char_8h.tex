A character type. 

The \mbox{\hyperlink{char_8h_a2c809211274ede9eddff5a750bdeb5ae}{char}} type represents a single character.

More specifically, since ‘character’ isn’t a well-\/defined concept in Unicode, \mbox{\hyperlink{char_8h_a2c809211274ede9eddff5a750bdeb5ae}{char}} will be a ‘\+Unicode scalar value’ in the future, when UTF-\/8 validation is fully supported.
