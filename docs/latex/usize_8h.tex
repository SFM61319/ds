The size of this type, \mbox{\hyperlink{usize_8h_ae96c519acbabee86b8f60110a91553a7}{DS\+\_\+\+USIZE\+\_\+\+BYTES}}, is how many bytes it takes to reference any location in memory. For example, on a 32 bit target, this is 4 bytes and on a 64 bit target, this is 8 bytes.
